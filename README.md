# SpamShield
Accurate spam classification achieving 97% accuracy using Multinomial Naive Bayes.

## Features
- 97% classification accuracy
- Real-time web interface
- Comprehensive NLP preprocessing
- Model comparison analysis

## Tech Stack
- Python, Scikit-learn, NLTK
- Streamlit for deployment
- Pandas, NumPy for data handling

## Results
- Accuracy: 97%
- Precision: 0.96
- Recall: 0.96
- F1-Score: 0.96
